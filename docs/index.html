Cyg-X-1

This website provides visualizations based on ASM and PCA data from the canonical black hole Cyg X-1. Human annotations (defining the hard, intermediate and soft state) are compared with Hiden Markov Models trained on ASM data.


Python code producing the figures:
-spectral shape
-human annotations scheme and associated streaming video
-HMM with 3 components
-HMM with 5 components
